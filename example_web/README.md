# React Calculator Web App Example

This directory demonstrates how **ai-behavior** schemas work in a React web application.

## About This Example

**Project:** Calculator Web App
**Stack:** React 18 + TypeScript + Vite
**Architecture:** Component-Based with Custom Hooks

### What's Demonstrated

1. **Project Manifest** (`ai_files/project_manifest.json`)
   - React/TypeScript tech stack
   - Component-based architecture
   - Modern web development practices

2. **Project Rules** (`ai_files/project_rules.json`)
   - React best practices
   - Component patterns
   - State management conventions

3. **Development Logbook** (`ai_files/logbooks/CALC-001.json`)
   - Web calculator implementation
   - React-specific development workflow

## Structure

```
example_web/
├── ai_files/

│   ├── components/           # React components
│   ├── hooks/                # Custom hooks

```
