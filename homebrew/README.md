# Homebrew Tap for Waves

Product development framework for the AI era — Claude, Codex, Gemini CLI.

## Installation

```bash
brew tap exovian-developments/waves
brew install waves
```

## Usage

```bash
# Initialize in a project
cd your-project
waves init claude

# Upgrade waves globally + update current project
waves upgrade

# Check version
waves --version
```

## Updating

```bash
waves upgrade
```
